\section{引言}
\label{sec:intro}

深度图像合成领域近年来取得了许多进展。目前达到最先进结果的是生成对抗网络（GAN），它能够以极快的速度合成高保真图像。但是，它们存在众所周知的问题，包括训练不稳定和模式崩溃，这导致样本多样性不足。解决这些问题仍然是一个开放的研究问题。

受到 Transformer~\cite{Vaswani17attention}和 GPT~\cite{gpt3}在自然语言处理中成功的启发，生成式 Transformer 模型在图像合成中获得了越来越多的关注~\cite{chen2020imagegpt,Esser21vqgan,Razavi19vqvae2}。通常，这些方法的目标是将图像建模为序列，并利用现有的自回归模型来生成图像。图像分两个阶段生成；第一阶段是将图像量化为一系列离散 token（或视觉词）。在第二阶段，学习一个自回归模型（例如 Transformer）根据先前生成的结果顺序生成图像 token（即自回归解码）。与 GAN 中使用的微妙的最小最大化优化不同，这些模型通过最大似然估计进行学习。由于设计上的差异，现有研究已经证明了它们相对于 GAN 的优势，提供了稳定的训练和改进的分布覆盖或多样性。

现有的生成式 Transformer 研究主要关注第一阶段，即如何量化图像以最小化信息损失，并共享从自然语言处理借来的相同第二阶段。因此，即使是最先进的生成式 Transformer~\cite{Esser21vqgan,Ramesh21dalle}仍然将图像幼稚地视为序列，其中图像被展平为按光栅扫描顺序（即从左到右逐行）的一维 token 序列（参见图~\ref{fig:decoding}）。我们认为这种表示对于图像既不是最优的也不是高效的。与文本不同，图像不是顺序的。想象一下艺术作品是如何创作的。画家从草图开始，然后通过填充或调整细节来逐步细化它，这与以前工作中使用的逐行打印形成了鲜明对比~\cite{chen2020imagegpt,Esser21vqgan}。此外，将图像视为平面序列意味着自回归序列长度以二次方增长，容易形成极长的序列——比任何自然语言句子都长。这不仅对长期相关性建模构成了挑战，也使解码变得难以处理。例如，用 GPU 自回归生成具有 32x32 个 token 的单个图像需要相当长的 30 秒。

本文介绍了一种用于图像合成的新型双向 Transformer，称为掩码生成图像 Transformer（\model）。在训练期间，\model 在类似于 BERT~\cite{Devlin19bert}中的掩码预测的代理任务上进行训练。在推理时，\model 采用了一种新颖的非自回归解码方法在恒定的步数内合成图像。具体地，在每次迭代中，模型同时并行地预测所有 token，但仅保留最有信心的 token。剩余的 token 被掩码，将在下一次迭代中重新预测。掩码比例逐渐降低，直到通过几次细化迭代生成所有 token。如图~\ref{fig:decoding}所示，\model 的解码速度比自回归解码快一个数量级，因为它只需要 8 步而不是 256 步来生成图像，并且每步内的预测是可并行化的。此外，双向自注意力不仅允许模型基于光栅扫描顺序中的先前 token 进行条件处理，还允许模型从所有方向的生成 token 生成新 token。我们发现掩码调度（即每次迭代中被掩码图像的比例）显著影响生成质量。我们提议使用余弦计划，并在消融研究中证实其有效性。

在 ImageNet 基准上，我们通过实验证明\model 在类条件生成 256$\times$256 和 512$\times$512 分辨率的图像时，既明显更快（快达 64 倍），也能生成比最先进的自回归 Transformer（即 VQGAN）更高质量的样本。与领先的 GAN 模型（即 BigGAN）和扩散模型（即 ADM~\cite{dhariwal2021diffusion}）相比，\model 提供可比的样本质量，同时产生更有利的多样性。值得注意的是，我们的模型在分类准确性得分（CAS）~\cite{Ravuri19CAS}和用于合成 512$\times$512 图像的 FID~\cite{FID}上建立了新的最先进技术。据我们所知，本文提供了第一个证据，证明了掩码建模对于在常见 ImageNet 基准上的图像生成的有效性。  

此外，MaskGIT 的多向性质使其易于扩展到对自回归模型来说很困难的图像操纵任务。图~\ref{fig:teaser}展示了一种类条件图像编辑的新应用，其中\model 根据给定的类重新生成边界框内的内容，同时保持上下文（框外）不变。这个任务对于自回归模型要么是不可行的，要么对于 GAN 模型很困难，但对于我们的模型来说很简单。
定量地，我们通过将\model 应用于图像修复和任意方向的图像外推来展示这种灵活性。
